# Adaptive Interpretation Context Refinement

> Interpretation context is assembled once at turn start and frozen for the entire planner loop. As the planner accumulates evidence, the frozen guidance becomes stale — leading to wasted steps, redundant investigation, and missed constraint adjustments. The system needs to refine its interpretation context adaptively based on evidence thresholds, context pressure, and thread transitions.
