# Unbounded Context Tier Model

> The system has access to effectively unlimited context across four tiers — inline artifacts, transit streams, sift indexes, and filesystem — but lacks formal traversal semantics connecting them. A truncated inline artifact cannot resolve to its full content in transit, related evidence in sift, or source files on disk. The tier model needs formalization with addressable locators and lazy resolution so the system can reach any depth on demand without holding everything in working memory.
