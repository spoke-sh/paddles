# Context Pressure And Relevance Signals

> Context budget exhaustion, staleness, and relevance decay are invisible to the capabilities framework. The system cannot report or respond to context pressure the same way it handles planner budget exhaustion. Context pressure and relevance need to be modeled as first-class capability signals so the system can self-diagnose and adapt when context quality degrades.
