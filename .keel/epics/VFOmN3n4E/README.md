# Recursive Self-Assessing Compaction

> Current context compaction is purely mechanical — fixed character limits with no awareness of relevance, staleness, or cross-component dependencies. The system needs to evaluate its own context state and decide what to compact, promote, archive, or surface using the same bounded planner/evidence mechanisms it uses for workspace tasks, with compacted summaries becoming first-class artifacts that can themselves be further compacted.
