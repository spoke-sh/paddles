# Transit-Native Context Addressing

> Context components rely on structural wiring at factory-assembly time and cannot discover or navigate to related context at runtime. Transit already provides lineage primitives (streams, branches, checkpoints) but paddles treats it as write-only. Components need addressable transit-backed locators so any context artifact can resolve related context through transit lineage rather than manual orchestrator assembly.
