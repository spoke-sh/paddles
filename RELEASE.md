# Release Process

This document outlines the release process for `paddles`.

## Versioning

`paddles` follows [Semantic Versioning 2.0.0](https://semver.org/).

## Checklist

- [ ] All tests are passing.
- [ ] The `keel` board is healthy (`keel doctor`).
- [ ] All relevant stories are accepted.
- [ ] The `CHANGELOG.md` is updated.
